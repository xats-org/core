## üéØ What is xats?

The **eXtensible Academic Text Standard (xats)** is a modern, JSON-based schema designed specifically for educational and academic content. Built from the ground up for the AI era, xats enables:

- ü§ñ **AI-Native Design**: Deeply semantic structure optimized for AI processing and generation
- üìö **Rich Educational Metadata**: Built-in support for learning objectives, assessments, and pedagogical structures
- ‚ôø **Accessibility First**: 100% WCAG 2.1 AA compliant with comprehensive accessibility features
- üîå **LMS Integration**: Native LTI 1.3 support for seamless Learning Management System integration
- üìä **Assessment Framework**: Comprehensive built-in assessment system with multiple question types
- üîê **Rights Management**: Comprehensive licensing and copyright management for publishers
- üìÇ **File Modularity**: Split large textbooks across multiple files for team collaboration
- üåç **Internationalization**: Full language and text direction support for global content
- üìñ **Advanced Indexing**: Semantic indexing with cross-references and hierarchical structure
- üöÄ **Extensible Architecture**: Domain-specific extensions without breaking compatibility

- Comprehensive alt text support
- Language identification for all content

## üöÄ Quick Start

### Installation

```bash
npm install -g @xats-org/core
```

### Validate a Document

```bash
xats-validate my-textbook.json
```

### Programmatic Usage

```javascript
import { validateDocument } from '@xats-org/core';

const document = {
  schemaVersion: "0.3.0",
  bibliographicEntry: {
    id: "calculus-101",
    type: "book",
    title: "Introduction to Calculus"
  },
  subject: "Mathematics",
  bodyMatter: {
    contents: [/* ... */]
  }
};

const { valid, errors } = await validateDocument(document);
```
